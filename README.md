# Quiz App

This is a simple quiz application built using React. It includes 5 multiple-choice questions, real-time answer selection, score tracking, and question navigation. It is designed to test basic knowledge with a clean and responsive UI.

## Technologies Used
- React
- JavaScript
- HTML
- CSS

## Features
- Multiple-choice quiz format
- Score tracking based on selected answers
- Simple and clean interface
- Responsive layout for desktop and mobile

## How to Run This Project Locally

If you want to run this project on your own computer, follow these steps:

Step 1: Install Required Software  
Make sure you have these installed:  
1. Node.js – Download from https://nodejs.org/  
2. Git – Optional but useful, from https://git-scm.com/downloads  

Check installations by running:  
node -v  
npm -v  
---
Step 2: Clone the Project from GitHub  
Open any folder on your PC  
Right-click → Git Bash Here (or open terminal)  
Run this command:  
git clone https://github.com/Anchalbisht10/quiz-app  

Then move into the project folder:  
cd quiz-app  
---
Step 3: Install Project Dependencies  
Run this command:  
npm install  
---
Step 4: Start the Development Server  
Run this command:  
npm start  
---
The app will start running in your browser at:  
http://localhost:3000  

If it doesn’t open automatically, just copy and paste that link into your browser.
